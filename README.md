\# Student Grade Tracker



\## 📌 Project Overview



Student Grade Tracker is a simple Java console application that helps manage student records and their grades.

The application allows users to add students, store their grades, and generate reports such as average grade, highest grade, and lowest grade.



This project demonstrates basic concepts of Java programming including object-oriented programming, collections, and file handling.



---



\## 🚀 Features



\### 1. Add Student



Users can add a new student with their name and grade.

The student information is stored in the application.



\### 2. View Student Report



Displays a list of all students along with their grades.



\### 3. Calculate Average Grade



The program calculates the average grade of all students stored in the system.



\### 4. Find Highest Grade



Identifies the student with the highest grade.



\### 5. Find Lowest Grade



Identifies the student with the lowest grade.



\### 6. File Storage



Student data is saved in a file (`Students.txt`) so that the data can be reused later.
